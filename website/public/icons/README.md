# PWA Icons

Please place the following icons in this directory:

- `icon-192.png`: 192x192 PNG icon.
- `icon-512.png`: 512x512 PNG icon.
- `badge-72.png`: 72x72 PNG badge (optional but used in sw.js).

These are required for the Car Launcher Pro Dashboard PWA and push notifications.
The current configuration expects these paths:
- `/icons/icon-192.png`
- `/icons/icon-512.png`
- `/icons/badge-72.png`
